# Spree CommerceThis is a repo for managing machine readable artifacts for the [Spree Commerce APIs](http://spreecommerce.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Spree Commerce APIs](http://spreecommerce.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://spreecommerce.com)- [Developer](http://spreecommerce.com)- [Twitter](https://twitter.com/spreecommerce)- [GitHub](https://github.com/spree)The [APIs.json](https://github.com/api-evangelist/spree-commerce/blob/master/apis.json) index provides links to all the relevant links for the Spree Commerce APIs, including the OpenAPI and Postman if they are present.
